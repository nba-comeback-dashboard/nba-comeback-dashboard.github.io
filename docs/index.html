NBA Comeback Dashboard#

Game data last updated: 04/21/2025.

The NBA Comeback Dashboard is a plotting tool that lets you analyze and compare NBA comeback probability data taken from the NBA play-by-play database from the 1996-1997 season to now.

Analysis

Looks at some trends in the data and tries to answer “Are big comebacks just much more common nowadays?” and other questions. Also includes a few notes on how this project was put together.

Dashboard

Provides a configurable plotting tool so you can slice and dice the data as you like (including for your favorite team).
