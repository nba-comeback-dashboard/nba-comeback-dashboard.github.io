About#

Feel free to email me or click here if you would like to tip this work so I can buy some more tokens.

This project started when I was trying to examine a rule of thumb about how safe leads are while watching NBA games and I wanted to create a chart I could reference when watching games.

While showing a friend some of the data, he suggested that I should try using AI to code the front end for this project since – in my day job writing scientific computing software – we aren’t allowed to use AI on our main codebase. So this was a perfect opportunity to experiment.
